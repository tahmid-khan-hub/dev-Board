Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are:
getElementById("id") – Selects an element by its ID.
getElementsByClassName("class") – Selects elements by class name (returns an HTMLCollection).
getElementsByTagName("tag") – Selects elements by tag name (returns an HTMLCollection).
querySelector("selector") – Selects the first element that matches the CSS selector.
querySelectorAll("selector") – Selects all elements that match the CSS selector (returns a NodeList).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML – Gets or sets the HTML content inside an element, including HTML tags.
innerText – Gets or sets only the visible text inside an element, ignoring hidden elements and extra spaces.
textContent – Gets or sets all text inside an element, including hidden text, but without parsing HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a phase in the DOM event propagation where an event starts from the target element and moves upward to its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement() and specify the tag name.
To add the element to the DOM, use methods like appendChild().
To remove an element, use remove() on the element itself or removeChild() on its parent.
